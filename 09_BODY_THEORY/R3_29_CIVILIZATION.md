# DEFINITION

文明とは、制度・技術・文化の集合ではない。
文明とは、ひとりひとりの身体状態が時間をかけて関係・場・社会へと波及し、構造として定着したものである。

すなわち文明とは、
**身体状態の総体が外部構造として現れた結果である。**

# STRUCTURE

文明は以下の階層構造で成立する。

1. ひとりの状態（身体）
2. 関係（相互作用）
3. 場（集団的状態）
4. 社会（持続構造）
5. 文明（長期的固定構造）

この階層は一方向ではなく、常に還流する。

身体 → 関係 → 場 → 社会 → 文明
↑　　　　　　　　　　　　　　↓
　　　観測・還流・再設計

# PRINCIPLE

文明は上から設計されるものではなく、
**身体状態から下から立ち上がる。**

・呼吸が浅い → 判断が速く浅くなる
・姿勢が固い → 関係が硬直する
・循環が失われる → 持続性が崩れる

逆に、

・呼吸が通る → 思考が深まる
・姿勢がほどける → 関係に余白が生まれる
・循環が保たれる → 文明が持続する

# EQUATION

文明状態 ＝ Σ（個体状態 × 関係密度 × 循環時間）

また、

文明の質 ＝ （位置 × 流動 × 周期）の集団平均

# OBSERVATION

文明は制度では観測できない。
文明は身体に現れる。

観測指標：

・呼吸の深さ（浅い / 通っている）
・姿勢の質（固定 / 流動）
・歩行の質（作業 / 周期）
・関係の圧（緊張 / 還流）
・時間の使われ方（断片 / 連続）

文明の歪みは、まず身体に現れる。

# CONNECTION

接続先：

・無意識性 → 操作の解除
・自然性 → 本来の秩序
・回帰 → 再通過
・実践 → 条件生成
・教育 → 次世代身体設計
・労働 → 持続構造
・共同体 → 関係構造
・未来 → 文明持続条件

文明は単体では存在せず、
必ず身体・関係・時間構造と接続している。
