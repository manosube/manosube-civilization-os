JAPANESE_TITLE: 回帰 ― 失われた身体ではなく、戻りうる身体

# DEFINITION

回帰とは、
失われた身体を再構築することではない。

回帰とは、
身体の内部にすでに存在している本来の秩序が、
再び通り始める現象である。

身体は壊れたのではない。
通路が閉じただけである。

回帰とは、
その通路が再び開かれることである。

# STRUCTURE

回帰は以下の構造を持つ。

1. 潜在構造の存在
   　身体には本来の秩序が保持されている

2. 通路の遮断
   　緊張・習慣・分断により通れなくなる

3. 条件の再構築
   　足裏・重力・呼吸・循環の回復

4. 局所回復の発生
   　小さな通りが生まれる

5. 波及的連鎖
   　局所回復が全身へ広がる

6. 全体再統合
   　身体が本来の循環構造へ戻る

回帰は「修復」ではなく、
「再通過」である。

# PRINCIPLE

回帰の原理は以下である。

* 身体の本来は消えない
* 機能は失われても構造は残る
* 通れないものは条件次第で再び通る
* 回復は足すことでなく、ほどくことで起こる
* 局所の通りは全体へ波及する

回帰とは、
新しく作ることではなく、
既にあるものを再び通すことである。

# EQUATION

回帰状態は以下で表される。

回帰状態 ＝ f（通路回復 × 循環再開 × 緊張解除）

または

回帰強度 ＝ Σ（局所通過 × 波及係数）

さらに

回帰可能性 ＝（潜在構造 - 固定度）

ここで重要なのは、
回帰は外部付与ではなく内部復元である点である。

# OBSERVATION

回帰は以下の兆候として観測される。

* 足裏の感覚が近づく
* 呼吸が自然に深まる
* 動作の努力が減少する
* 姿勢が力まず成立する
* 動きが連鎖し始める
* 意識の介入が減少する

重要なのは、
回帰は劇的変化としてではなく、
微細な変化の連鎖として始まることである。

# CONNECTION

回帰は以下と接続する。

* 無意識性
  　→ 操作を超えた自律的身体

* 自然性
  　→ 人為を超えた秩序

* 実践
  　→ 回帰条件の構築

* 循環
  　→ 回帰の持続構造

* 文明
  　→ 回帰の波及先

回帰は身体の終点ではなく、
文明への接続点である。
