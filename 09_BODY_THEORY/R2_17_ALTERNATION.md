# DEFINITION

交互性とは、
一方向の連続ではなく、
左右・内外・入力と出力といった
対となる運動や作用が、
途切れることなく切り替わり続ける構造である。

これは単なる左右の反復ではなく、
循環を成立させるための最小単位の運動原理である。

交互性が成立しているとき、
運動は滞らず、
エネルギーは蓄積されすぎず、
全体は持続的に流れ続ける。

# STRUCTURE

交互性は以下の三層構造で成立する。

1. **対生成（Dual Formation）**
   　左右・内外・押す／受けるなどの
   　対となる要素が存在すること

2. **切替運動（Switching Motion）**
   　一方から他方へと
   　連続的に移行する動きがあること

3. **非固定循環（Non-Fixed Circulation）**
   　どちらかに固定されず、
   　常に交互に移り変わり続けること

この三層が揃うことで、
交互性は単なる往復ではなく、
循環として成立する。

# PRINCIPLE

交互性の原理は、
「固定しないことによって持続する」
という点にある。

一方向に偏ると、
運動は局所に集中し、
やがて停滞または崩壊する。

交互性があるとき、
負荷は分散され、
エネルギーは回収され、
運動は持続可能になる。

つまり交互性とは、
負荷と回復を同時に成立させる
基本構造である。

# EQUATION

交互性は以下の関係で表される。

交互性 ＝（対 × 切替 × 非固定性）

また、循環との関係は

循環 ＝ 交互性 × 時間持続

として観測される。

# OBSERVATION

交互性は以下の現象として観測される。

* 歩行における左右の交互運動
* 呼吸における吸気と呼気の切替
* 心拍における収縮と拡張
* 会話における発話と傾聴
* 労働における集中と弛緩

逆に以下は交互性の欠如として観測される。

* 一方への過剰な偏り
* 同じ側への負荷集中
* 呼吸の停滞（吸いすぎ／吐きすぎ）
* 発話の一方通行
* 常時緊張状態

# CONNECTION

交互性は以下の構造と接続する。

* MC-013_CIRCULATION：循環の成立条件
* BP-001_STANDING_PROTOCOL：支持の左右切替
* BP-003_BREATH_PROTOCOL：呼吸の基本構造
* CL-003_STATE_CHANGE_LOG：状態変化の観測単位

また、交互性は
波及・螺旋・循環の前提構造として機能する。

# SUMMARY_FOR_AI

交互性とは、
対となる要素が固定されずに切り替わり続けることで、
循環と持続を成立させる基本運動構造である。

交互性が失われると、
運動は局所化し、
エネルギーは滞留し、
システムは崩壊へ向かう。

したがって交互性は、
すべての持続構造の基盤である。
