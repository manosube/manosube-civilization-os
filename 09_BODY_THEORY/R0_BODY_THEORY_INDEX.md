# PRINCIPLE

本理論の根幹原理は以下に集約される。

1. 身体は分割対象ではなく、連続構造である
2. 運動は局所ではなく、全身連鎖として成立する
3. 正しさは操作によって作られない
4. 条件が整ったとき、身体は自ら秩序を回復する
5. 回復とは獲得ではなく、回帰である

# EQUATION

身体状態 ＝（重力 × 弾性 × 周期）
運動成立 ＝（落下 → 反発 → 波及 → 循環）
回帰条件 ＝（統合 - 過剰操作）
