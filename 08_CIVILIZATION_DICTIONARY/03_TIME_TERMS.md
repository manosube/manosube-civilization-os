# DEFINITION

TIME_TERMSとは、
文明を構成する「時間の性質」を観測・記述するための語彙体系である。

ここでいう時間とは、
単なる計測単位としての時間ではない。

時間とは、
状態が変化し続ける中で現れる
循環・固定・断絶・回帰という運動構造そのものである。

時間は数値ではなく、
通り方として観測される。

# STRUCTURE

TIME_TERMSは、三つの基底構造によって成立する。

1. 循環時間（CIRCULATION TIME）
2. 固定時間（FIXED TIME）
3. 断絶時間（DISCONTINUITY TIME）

循環時間は流れを生み、
固定時間は滞留を生み、
断絶時間は連続性を破壊する。

この三要素の関係によって、
文明の時間構造が決定される。

時間は一本の直線ではない。
時間は状態構造の現れである。

# PRINCIPLE

時間の本質は、流速ではなく循環性にある。

・循環する時間は、回復と持続を生む
・固定された時間は、劣化と停滞を生む
・断絶した時間は、崩壊と不連続を生む

文明は時間の長さでは持続しない。
文明は時間の構造によって持続する。

時間とは、
状態が戻りうるかどうかという
回帰可能性の指標である。

# EQUATION

文明耐久性 ＝（循環時間 - 固定時間）

文明崩壊速度 ＝（断絶時間 × 固定時間）

文明持続性 ＝（循環時間 ÷ 総時間）

時間は均一ではない。
時間は分布し、偏在し、構造化される。

時間の質が、文明の質を決定する。

# OBSERVATION

以下の兆候は、時間構造の歪みを示す。

・同一状態の長期停滞（固定時間の蓄積）
・変化の断続的分断（断絶時間の発生）
・回復不能状態（循環時間の消失）
・常時の時間圧迫（圧縮時間の増大）
・過去と現在の非接続（連続性の崩壊）

一方で、

・変化が滑らかに連鎖する
・状態が自然に戻る
・周期とリズムが存在する
・無理なく持続する

これらは循環時間が機能している状態である。

# CONNECTION

TIME_TERMSは以下と接続する。

・CIVILIZATION_STATE
→ 状態変化として時間が現れる

・CIRCULATION
→ 時間構造の基盤

・COLLAPSE_STRUCTURE
→ 固定時間と断絶時間の蓄積結果

・RETURN_PROTOCOL
→ 循環時間の再生成プロセス

・LOG_STRUCTURE
→ 時間構造の記録と可視化

時間は単独では存在しない。
時間は常に状態と結合している。
