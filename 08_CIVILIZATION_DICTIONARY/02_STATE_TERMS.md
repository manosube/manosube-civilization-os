# DEFINITION

STATE_TERMSとは、
文明を構成する「状態」を観測・記述するための基本語彙群である。

ここでいう状態とは、
物理的な位置や配置だけでなく、
流れ・変化・持続・緊張・循環といった
時間的・関係的・構造的な性質を含む。

文明は固定された構造ではなく、
状態の連続的な変化として存在する。

したがって、
STATE_TERMSは
文明を「もの」ではなく
「変化し続ける状態」として扱うための
最小単位の観測言語である。

# STRUCTURE

STATE_TERMSは以下の三層構造を持つ。

1. 基底状態語彙
   　位置・流動・周期など、文明状態を構成する基本軸

2. 関係状態語彙
   　接触・還流・緊張・境界など、状態間の関係性

3. 変化状態語彙
   　崩壊・修復・波及・転移など、状態の時間的変化

この三層は分離されない。
すべては同時に観測されるべきものであり、
文明状態はこれらの重なりとして現れる。

# PRINCIPLE

文明は「状態の集合」ではない。
文明は「状態の運動」である。

状態は固定されない。
状態は常に変化し、連鎖し、還流する。

したがってSTATE_TERMSは、
定義を固定するためではなく、
変化を観測するために存在する。

状態を固定した瞬間、
文明は死に始める。

状態を流動として捉えるとき、
文明は持続する。

# EQUATION

文明状態 ＝（位置 × 流動 × 周期）

位置：どこにあるか
流動：どのように動いているか
周期：どのリズムで変化しているか

この三軸が揃うとき、
状態は単なる断片ではなく
「生きた文明状態」として成立する。

# OBSERVATION

STATE_TERMSの観測は以下の問いによって行う。

・いまこの状態はどこにあるか（位置）
・何が流れ、何が止まっているか（流動）
・どのようなリズムで繰り返されているか（周期）

さらに、

・どこに緊張があるか
・どこで循環が途切れているか
・どこで過剰な固定が起きているか

を同時に観る。

STATE_TERMSは、
「良い／悪い」を判断するための語彙ではない。

STATE_TERMSは、
「いま何が起きているか」を
歪みなく観るための語彙である。

# CONNECTION

STATE_TERMSは以下と接続する。

・THREE_PRINCIPLES（重心・動作・呼吸）
・CIRCULATION（循環構造）
・OBSERVATION（観測構造）
・LOG_STRUCTURE（文明ログ）
・CIVILIZATION_ENERGY（文明エネルギー）

特に重要なのは、
STATE_TERMSがすべての上位概念の基盤であることである。

状態が観測できなければ、
循環も、崩壊も、修復も、設計も成立しない。
