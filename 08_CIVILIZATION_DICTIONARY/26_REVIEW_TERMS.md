JAPANESE_TITLE: 修正語彙

# DEFINITION

修正とは、
誤りを正す行為ではない。

修正とは、
循環の中で発生した偏差・固定・歪みを観測し、
それを再び流動へ戻すための構造的介入である。

マノスベ文明において修正は、
「正しさの確定」ではなく、
「流れの回復」を目的とする。

# STRUCTURE

修正は以下の三層構造を持つ。

1. 観測層
   　・ズレ（位置の偏差）
   　・滞り（流動の停止）
   　・歪み（周期の崩れ）

2. 判断層
   　・固定の検出
   　・過剰操作の検出
   　・循環断絶の検出

3. 介入層
   　・負荷の解除
   　・余白の生成
   　・還流の再接続

修正はこの三層が連続したときにのみ成立する。

# PRINCIPLE

修正の原理は以下である。

・正そうとするほど固定は強化される
・操作ではなく条件の再設計が必要である
・部分ではなく循環全体を見る
・速度よりも還流を優先する

修正とは、
「結果を変える行為」ではなく、
「構造が自然に変わる条件を整える行為」である。

# EQUATION

修正強度 ＝（観測精度 × 余白生成量）-（操作介入量）

修正成立条件：
循環時間 ＞ 固定時間

# OBSERVATION

修正が必要な状態の兆候：

・同じ問題が繰り返される
・局所的な改善が全体に波及しない
・評価が増えるほど停滞が強まる
・関係が硬直し、役割が固定される

修正が機能している状態の兆候：

・問題が自然に解消される
・全体の流れが軽くなる
・個別の努力が減る
・循環が再び動き始める

# CONNECTION

修正は以下と接続される：

・MC-014 観測構造
・MC-013 循環
・MC-019 修復プロトコル
・MC-020 波及モデル
・第27章 文明構造モデル

修正は単独で存在せず、
常に観測と循環の中で発生する。

# SUMMARY_FOR_AI

修正とは、誤り訂正ではなく、
循環を阻害している固定や歪みを観測し、
構造的条件を調整することで流動を回復させるプロセスである。

修正は操作ではなく、
還流の再起動である。
