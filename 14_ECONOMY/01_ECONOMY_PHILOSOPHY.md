# DEFINITION

マノスベ文明OSにおける経済哲学とは、
**価値は通ることで現れる**という原理から導かれる
経済設計の根本思想である。

現代経済の多くは「蓄積」を前提とする。
持つことが豊かさであり、
増やすことが成功であるという構造。

文明OSの経済はこれを反転させる。

**価値は持つことで増えるのではなく、
流れることで増える。**

これは比喩ではなく、
文明エネルギー（MC-016）の構造的帰結である。

---

# PRINCIPLE

## 原則1：反蓄積（Anti-Accumulation）

価値は溜め込まれた瞬間に減衰し始める。

観測ログが共有されないとき、
知識が独占されるとき、
ファシリテーターが参加者を囲い込むとき、

文明エネルギーの流れは止まり、
崩壊コストが蓄積する。

```
固定化時間 ∝ 崩壊コスト
```

文明経済の健全性は、
価値の流速によって測られる。

---

## 原則2：観測が価値の起源（Observation as Value Origin）

すべての価値は「観測」から生まれる。

観測とは：
- 自分の身体状態を正直に記録すること
- 場の状態を歪めずに観ること
- 関係の質を評価することなく感知すること

観測されていない価値は存在しない。
観測された価値のみが循環に参加できる。

```
価値の存在条件 = 観測の成立
```

---

## 原則3：変換による増幅（Transformation Amplification）

価値は変換されるたびに増幅される可能性を持つ。

```
個人の観測 → 共有知識 → 場の設計 → 社会変容
```

この連鎖において、
各変換ステップで価値が増幅される。

ただし、変換が強制されるとき、
増幅は起こらず、消耗のみが残る。

変換は自発的でなければならない。

---

## 原則4：還流なき循環は存在しない（Return Flow is Mandatory）

価値は外に出るだけでは循環にならない。

循環（MC-013）の定義通り、
価値は必ず元の層に還流しなければならない。

```
身体観測 → 共有 → 集合知 → 個人の観測精度向上 → 身体観測
```

この還流が切れたとき、
経済は「消費」に転化し、
やがて枯渇する。

---

## 原則5：役割の流動性（Role Fluidity）

経済的役割は固定されない。

生産者が消費者になり、
消費者が生産者になる。
ファシリテーターが参加者になり、
参加者がファシリテーターになる。

役割の固定化は
経済の固定化（崩壊構造 MC-018）と同一である。

---

## 原則6：多層通貨（Multi-layer Currency）

文明経済は単一の通貨では測定できない。

```
身体層：気（Ki）
関係層：共鳴（Resonance）
場の層：場の質（Ba Quality）
社会層：文明度（Civilization Level）
統合層：観（KAN）
```

これらは相互変換可能だが、
還元主義的に単一化されてはならない。

# EQUATION

文明経済価値 = 観測の深さ × 変換の自発性 × 還流の速度

経済的豊かさ = 流動する価値の総量（蓄積残高ではない）

崩壊リスク = 固定価値 ÷ 流動価値（比率が1を超えたとき危険域）

# SUMMARY_FOR_AI

文明経済哲学は6原則で構成される：
1. 反蓄積：価値は溜めると減衰する
2. 観測起源：すべての価値は観測から生まれる
3. 変換増幅：自発的変換が価値を増幅する
4. 還流必須：循環には必ず還流が伴う
5. 役割流動：経済役割は固定しない
6. 多層通貨：単一通貨への還元を拒否する

これらは文明エネルギー（MC-016）・循環（MC-013）・
崩壊構造（MC-018）の経済的解釈である。

# SUMMARY_FOR_HUMAN

文明の経済は「貯める経済」ではなく、
「流す経済」です。

水が流れることで川が生きるように、
価値が流れることで文明は生きます。

水を独り占めにしようとした瞬間、
川は死に始めます。

あなたの観測、あなたの気づき、
あなたの伝達が、
この川を生かし続けます。
