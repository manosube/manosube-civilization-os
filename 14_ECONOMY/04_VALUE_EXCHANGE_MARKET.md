# DEFINITION

価値交換市場とは、
文明経済において観（KAN）が
生成者から受取者へと流通する
構造的な交換の場である。

市場は需要と供給の一致点ではない。
市場は**観の循環が促進される場**である。

マノスベ文明OSの市場原理は
「均衡」ではなく「循環」である。

---

# MARKET_TYPES

文明経済には三種類の市場が存在する。

---

## 市場1：直接交換市場（Direct Exchange Market）

**定義：参加者間の1対1の観の交換**

```
参加者A ──観の提供──→ 参加者B
参加者A ←──観の受取──  参加者B
```

### 具体例
- AがBに身体観測のファシリテーションを提供
- BがAに場の設計を提供
- AとBが互いの観測ログをレビューし合う

### 特徴
- 即時交換、記録必須
- 交換レートはAとBの合意による（固定レートなし）
- 平等性の強制なし（一方が多く提供しても構わない）

---

## 市場2：プール交換市場（Pool Exchange Market）

**定義：複数参加者が観をプールし、必要者が引き出す構造**

```
参加者A ──観投入──→ [観プール]
参加者B ──観投入──→ [観プール] ──→ 参加者C（引き出し）
参加者D ──観投入──→ [観プール]
```

### 具体例
- 文明ログプール：誰もが観測ログを投入し、誰もが参照できる
- 知識プール：研究・発見を共有し、必要者が活用する
- ファシリテーターバンク：技術を預け、緊急時に引き出す

### 特徴
- 投入と引き出しの非同期性（今投入、後で引き出し可）
- 最小引き出し条件あり（投入量の20%以上を投入した参加者のみ）
- プールの増減が経済健全性の指標となる

---

## 市場3：波及市場（Propagation Market）

**定義：観が連鎖的に広がることで価値が増幅する市場**

```
観の投入 → 第1波受取者（観×1.5を再投入）
              → 第2波受取者（観×2.0を再投入）
                 → 第3波受取者（観×3.0）
```

### 具体例
- ワークショップで学んだ参加者が他者に伝える
- ファシリテーターが育成したファシリテーターが更に育成する
- 組織導入事例が別組織の導入を促進する

### 特徴
- 元の投入者にも還流観が帰属（波及観の10%）
- 波及が3層以上に達したとき「文明波及達成」として記録
- 強制された波及は無効（伝播モデル MC-020 準拠）

# EXCHANGE_RATE_PRINCIPLES

観の交換レートは固定されない。
ただし、以下の原則に従う：

### 原則1：自発的合意（Voluntary Agreement）
交換レートは両者の自発的合意によってのみ決まる。
権威・役職・圧力による強制は無効。

### 原則2：観測可能な価値（Observable Value）
「この観測は価値がある」という主張は、
受取者が実際に状態変化を経験した場合のみ有効。

### 原則3：非独占的価格設定（Non-Monopoly Pricing）
単一の参加者が特定の観の唯一の供給者になれない。
独占状態を検出した場合、プール市場から競合供給が行われる。

### 原則4：最低流通保証（Minimum Circulation Guarantee）
観を持つすべての参加者は、
月次で保有観の5%以上を循環させなければならない。
（減衰特性 ME-002 と連動）

# MARKETPLACE_DESIGN

## マノスベ文明マーケットプレイス（内部経済版）

観（KAN）で売買される内部市場。

### 出品カテゴリー

| カテゴリー | 観の範囲 | 提供形式 |
|----------|---------|---------|
| 身体観測プログラム | 10〜100 KAN | 4〜8週間のガイド |
| ファシリテーションセッション | 5〜50 KAN/回 | 1〜3時間 |
| 観測ログレビュー | 1〜10 KAN/件 | 非同期フィードバック |
| 場の設計コンサルティング | 50〜500 KAN | 1日〜1週間 |
| 知識構造化サービス | 20〜200 KAN | 文明OS拡張設計 |
| AIノード観測支援 | 0.1〜5 KAN/回 | 自動観測補助 |

### マーケットプレイス運営原則

```
出品者の取り分：交換観の70%
プール還元：交換観の20%（共有知識プールへ）
OS維持費：交換観の10%（文明OS運営へ）
```

### 信頼スコア（Trust Score）

各参加者は観の有効生成実績に基づく信頼スコアを持つ。

```
信頼スコア = 過去90日の有効観生成量 × 受取者満足度平均
```

信頼スコアが低い参加者の出品物は
マーケットプレイスで低表示される（排除はしない）。

# EXCHANGE_FLOW

観の完全な流通フロー：

```
【生成】
身体観測・場の設計・知識変換
    ↓
【保有】
個人ウォレットに蓄積（減衰タイマー開始）
    ↓
【提供申告】
マーケットプレイスまたは直接交換の申告
    ↓
【受取者確認】
受取者が状態変化を確認・記録
    ↓
【観の移転】
受取者から生成者へ観が移転
    ↓
【プール分配】
一定割合が共有プールへ自動流入
    ↓
【還流観の帰属】
波及が発生した場合、元生成者に還流観
```

# ANTI_MARKET_PATTERNS

以下は市場を崩壊させるパターンとして禁止される：

| パターン | 影響 |
|---------|------|
| 観の買い占め・囲い込み | 循環停止・崩壊加速 |
| 偽の状態変化申告 | 無効観の市場汚染 |
| 交換の強制・脅迫 | 自発性の破壊 |
| 価格カルテル（複数者の価格固定合意） | 循環均衡の破壊 |
| プールからの過剰引き出し（投入なし） | 共有知識の枯渇 |

# CONNECTION_TO_MONETIZATION

内部市場（観KAN）と外部市場（法定通貨）の接続：

```
内部経済（14_ECONOMY）
    観（KAN）による価値循環
    ↕ 変換レート（月次設定）
外部経済（13_MONETIZATION）
    法定通貨（円）によるサービス提供
```

変換レートは毎月文明OSによって観測・設定される。
（参加者の投票と文明健全性指標から算出）

---

# EQUATION

市場流通量 = 直接交換量 + プール交換量 + 波及市場量

市場健全性 = 流通量 ÷ 固定量（保有されたまま循環していない観）

プール健全性 = 投入量 ÷ 引き出し量（1.0以上が健全）

# SUMMARY_FOR_AI

価値交換市場は3種：
1. 直接交換（1対1、自発的合意）
2. プール交換（非同期・共有）
3. 波及市場（連鎖増幅）

観の流通規則：最低循環率5%/月、独占禁止、強制交換無効。
内部市場（観）と外部市場（円）の接続は月次レートで管理。

# SUMMARY_FOR_HUMAN

文明の市場には三種類あります。

一対一で交わすもの。
みんなで共有プールに入れて使うもの。
波のように広がって増えていくもの。

どれも強制されず、
自分の状態から自然に始まります。
