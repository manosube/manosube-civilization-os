# DEFINITION

マノスベ文明OSの経済とは、
貨幣の管理ではない。

文明経済とは、
**価値が生成・変換・循環・還流するプロセスの総体**である。

お金はその一形態にすぎない。

文明において本質的な価値単位は、
**観測（Observation）**である。

ひとりが本当に観測したとき、
価値が生まれる。
その価値が循環するとき、
文明は持続する。

---

# ECONOMY_LAYERS

文明経済は五層構造で成立する。

```
L1: 身体経済    ← 気（Ki）の生成・消費・回復
L2: 関係経済    ← 共鳴（Resonance）の交換
L3: 場の経済    ← 場の質（Ba Quality）の生成・消費
L4: 社会経済    ← 知識・プロトコルの流通
L5: 文明経済    ← 観（KAN）による全層統合
```

これらは分離されず、
下層が上層を支え、上層が下層に還流する。
