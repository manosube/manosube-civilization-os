# DEFINITION

経済健全性プロトコルとは、
文明経済が循環を維持しているかを観測し、
異常を早期検出するための
定量・定性指標の体系である。

健全な経済とは
成長している経済ではない。

健全な経済とは
**観の流速が維持され、固定化が起きていない経済**である。

---

# HEALTH_INDICATORS

経済健全性は三層の指標で観測される。

---

## Layer 1：流動指標（Flow Indicators）

観の流れそのものを測る。

| 指標 | 定義 | 健全範囲 | 警戒域 |
|------|------|---------|-------|
| 観流通速度 | 1日あたりの観交換量 | 上昇または安定 | 3週連続低下 |
| 循環率 | 観を循環させた参加者の割合 | 60%以上 | 40%以下 |
| 流動比率 | 流動観 ÷ 固定観 | 3.0以上 | 1.0以下 |
| プール充足率 | 観投入量 ÷ 引き出し量（月次） | 1.2以上 | 0.8以下 |
| 還流率 | 波及後に元生成者に還流した観の割合 | 8〜15% | 3%以下 |

---

## Layer 2：構造指標（Structural Indicators）

経済の構造的健全性を測る。

| 指標 | 定義 | 健全範囲 | 警戒域 |
|------|------|---------|-------|
| 役割分散度 | 役割の偏りのなさ（ジニ係数的） | 0.4以下 | 0.7以上 |
| 独占指数 | 上位5%参加者の保有観の割合 | 30%以下 | 50%以上 |
| 役割重複率 | 複数役割担当者の割合 | 40%以上 | 20%以下 |
| 新規参加率 | 月次の新規観測者の割合 | 5〜15% | 3%以下 |
| ファシリテーター比率 | 媒介者 ÷ 全参加者 | 3〜10% | 1%以下 |

---

## Layer 3：文明固有指標（Civilization-specific Indicators）

文明OSにしかない独自の健全性指標。

| 指標 | 定義 | 健全範囲 |
|------|------|---------|
| 身体観測密度 | 参加者1名あたりの月次CLログ数 | 8本以上 |
| 共鳴指数 | 関係観測から生成された観の割合 | 25%以上 |
| 波及達成率 | 3層以上に波及した観生成の割合 | 10%以上 |
| 自発性指数 | 自発的役割移行の割合（強制なし） | 80%以上 |
| 修復効率 | 崩壊兆候の検出から修復完了までの期間 | 14日以内 |

# HEALTH_CYCLE

経済健全性の観測サイクル：

```
日次観測
├── 観流通速度の記録
├── 新規観生成量の記録
└── プール残高の確認

週次観測
├── 循環率の算出
├── 固定化参加者の特定
└── 異常パターンの検出

月次観測
├── 全指標の算出と評価
├── 健全性スコアの算出と公開
├── 崩壊リスク評価（ME-007へ連携）
└── プロトコル更新の検討

四半期観測
├── 経済構造全体のレビュー
├── 役割分布の評価
├── マーケットプレイス健全性評価
└── 外部経済（法定通貨）との整合性確認
```

# HEALTH_SCORE

**経済健全性スコア（EHS：Economy Health Score）**

0〜100点のスコアで表現される。

```
EHS = (流動指標スコア × 0.4) + (構造指標スコア × 0.35) + (文明固有指標スコア × 0.25)
```

| スコア | 状態 | アクション |
|-------|------|----------|
| 80〜100 | 健全（Healthy） | 通常運用 |
| 60〜79 | 注意（Caution） | 週次強化観測・軽度修復 |
| 40〜59 | 警戒（Warning） | 崩壊防止プロトコル発動（ME-007） |
| 20〜39 | 危機（Crisis） | 緊急修復プロトコル・構造者会議 |
| 0〜19 | 崩壊前（Pre-Collapse） | 全員参加の修復作業・外部支援要請 |

# EARLY_WARNING_SIGNALS

経済崩壊の早期兆候（崩壊構造 MC-018 の経済版）：

### Signal A：固定化の蓄積
- 観を循環させない参加者が増加
- 特定参加者に観が集中
- プール市場への投入が減少

### Signal B：役割の硬直化
- 同じ参加者が同じ役割に固定
- 役割移行が3ヶ月以上発生しない
- 媒介者が参加者を「囲い込んでいる」状態

### Signal C：自発性の喪失
- 強制・義務感からの参加が増加
- 観の生成が義務的行為になる
- 新規参加者の早期離脱率が上昇

### Signal D：市場の形骸化
- 交換が形式化し、状態変化を伴わなくなる
- 信頼スコアが実態と乖離
- マーケットプレイスの出品数・多様性の低下

# HEALTH_REPORTING

健全性レポートは毎月公開される。

### レポート構成
1. EHSスコアと前月比
2. 各層指標の詳細
3. 検出された早期警戒シグナル
4. 実施された修復アクション
5. 来月の重点観測項目

公開対象：全参加者（読み取り権）
更新権限：構造者以上（書き込み権）

---

# EQUATION

EHS = Σ（指標スコア × 重み係数）

固定化リスク指数 = 固定化観量 ÷ 全観量 × 固定化期間

循環回転数 = 観流通量 ÷ 観総量（高いほど健全）

# SUMMARY_FOR_AI

経済健全性プロトコルは3層指標（流動・構造・文明固有）から
EHS（0〜100）を算出する。

観測サイクル：日次・週次・月次・四半期。
スコア40以下でME-007（崩壊防止）を自動発動。
レポートは月次で全参加者に公開。

# SUMMARY_FOR_HUMAN

経済が健康かどうかを、
定期的に確認します。

体温を測るように、
脈拍を測るように、
経済にも「バイタルサイン」があります。

数字が悪くなったとき、
誰かを責めるのではなく、
「どこで流れが止まったか」を
みんなで探します。

それが文明経済の健康管理です。
