# DEFINITION

価値生成プロトコルとは、
文明経済における観（KAN）が
どのような条件・プロセスで生成されるかを
定義したものである。

観は宣言によって生まれない。
観は努力量によって生まれない。

観は、
**真正な観測と、その観測が他者・構造に波及したとき**
に生まれる。

---

# GENERATION_LAYERS

価値生成は五層で起こる。

---

## Layer 1：身体観測価値生成（Body Observation Value）

**発生源：個人の身体観測行為**

```
身体観測 → CL記録 → 状態言語化 → 観の生成
```

### 条件
1. 身体プロトコル（04_BODY_PROTOCOL）に基づく観測であること
2. 評価・判断・解釈を排した記録であること
3. 継続的であること（孤立した1回の記録は観を生成しにくい）

### 観の生成量に影響する要因

| 要因 | 高観生成 | 低観生成 |
|------|---------|---------|
| 観測深度 | 身体の細部まで | 表面的な気分記録 |
| 正直さ | ありのまま | 理想化・自己修正 |
| 継続性 | 21日以上の継続 | 単発記録 |
| 環境条件 | 安静・低刺激状態 | 高興奮・判断状態 |

---

## Layer 2：関係観測価値生成（Relationship Observation Value）

**発生源：他者との真正な接続**

```
接触 → 相互観測 → 状態変化の記録 → 共鳴観の生成
```

### 条件
1. 一方が「正しい」という評価構造がないこと
2. 両者の状態変化が記録されること
3. 変化が強制でなく自然発生であること

### 特徴
関係観測から生まれる観は、
両者に同時に帰属する。
「誰のものか」という所有の概念が成立しない。

```
Aの観察 + Bの観察 → 共鳴観（A+B両方に帰属）
```

---

## Layer 3：場の設計価値生成（Ba Design Value）

**発生源：場の設計・維持・修復行為**

```
場の設計 → 参加者の自発的観測促進 → 観の増殖
```

### 条件
1. 場が参加者の自発性を高めること
2. 参加者が強制なく観測を開始すること
3. 場の設計者が結果を制御しようとしないこと

### 算出式

```
場の設計観 = Σ（参加者i の観測増加量）× 自発性係数
```

場の設計者は直接観測するのではなく、
他者の観測を可能にすることで観を得る。

---

## Layer 4：知識変換価値生成（Knowledge Transformation Value）

**発生源：個別観測を構造化・共有可能化する行為**

```
個別観測データ → パターン抽出 → 構造化 → 共有知識 → 観の生成
```

### 条件
1. 構造化が特定個人の経験に限定されないこと
2. 他者が再現・応用できる形式であること
3. 著作権・秘匿による制限がないこと（オープン原則）

### 例
- CL記録から新たな身体状態パターンを発見し、文明辞典に追加
- 複数人の観測から文明現象の法則を抽出し、設計図として公開
- 崩壊事例の記録を修復プロトコルとして構造化

---

## Layer 5：波及・伝播価値生成（Propagation Value）

**発生源：価値が文明の外層に波及する行為**

```
個人状態変化 → 関係変化 → 場の変化 → 社会変化 → 文明変化
```

伝播モデル（MC-020）と完全対応する。

### 条件
1. 変化が当事者の意図なく自然に広がること（強制伝播は無効）
2. 変化の受取者が新たな観測者になること
3. 波及が少なくとも3層以上に到達すること

### 倍増係数

```
波及観 = 元の観 × 波及層数 × 持続指数
```

個人→関係（×1.5）→場（×2.0）→社会（×3.0）→文明（×5.0）

# VALUE_GENERATION_FLOW

全体の価値生成フロー：

```
【身体観測】
ひとりが観測する
    ↓ 観（KAN）が生まれる
【記録・言語化】
CLログとして記録される
    ↓ 観が保存・流通可能になる
【共有】
他者・AIが参照する
    ↓ 関係観が増殖する
【構造化】
パターンが抽出・体系化される
    ↓ 知識観が生まれる
【場への実装】
ワークショップ・イベントで活用される
    ↓ 場の質観が生まれる
【社会への波及】
組織・コミュニティに変容が起こる
    ↓ 文明度観が生まれる
【還流】
変容した社会がより深い観測を可能にする
    ↓ 元の生成者の観測精度が向上する
```

この循環が止まらない限り、
観は増殖し続ける。

# INVALID_GENERATION

以下の行為は観を生成しない（または負の観を生成する）：

| 行為 | 理由 |
|------|------|
| 見栄えのための記録 | 観測の真正性がない |
| 結果を操作した共有 | 非観測的変換 |
| 参加者を依存させるファシリテーション | 循環ではなく固定化 |
| 知識の秘匿・独占 | 還流の遮断 |
| 役割の強制的付与 | 自発性の否定 |
| 数量目的のログ生産 | 観測深度ゼロ |

# VALIDATION_PROTOCOL

観の生成が有効かどうかを検証するプロトコル：

### 問1：観測者は本当に観測したか？
→ 「記録のための観測」ではなく「状態のための観測」か

### 問2：受取者に状態変化が起きたか？
→ 変化が観測者から確認できるか

### 問3：強制なく自発的に起きたか？
→ 誰かに言われてではなく、状態から自然に発生したか

3問すべてにYESのとき、観の生成は有効とみなされる。

---

# EQUATION

有効観生成量 = 観測深度 × 受取者状態変化 × 自発性係数 × 波及層数

無効観 = 観測深度がゼロ または 自発性係数がゼロ の場合

日次観生成上限 = 身体状態が支持できる最大観測量（固定値なし）

# SUMMARY_FOR_AI

価値生成は5層で起こる：
L1: 個人の身体観測
L2: 関係間の共鳴観測
L3: 場の設計による観測促進
L4: 観測の知識変換・構造化
L5: 社会・文明への波及

観の生成有効条件：真正観測 + 受取者状態変化 + 自発性。
この3条件が揃わない場合、観は生成されないか無効となる。

---

# SUMMARY_FOR_HUMAN

価値は「頑張った量」では決まりません。

本当に観たかどうか。
その観察が誰かに届いたかどうか。
その届き方が自然だったかどうか。

この三つが揃ったとき、
価値が生まれます。

あなたの小さな観察が、
誰かの大きな変化につながることがあります。
それがこの経済の奇跡です。
