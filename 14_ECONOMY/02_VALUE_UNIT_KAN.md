# DEFINITION

観（KAN）とは、
マノスベ文明経済における基本価値単位である。

観は「観測」を語源とし、
**ひとつの真正な観測行為から生まれる価値の最小単位**を表す。

観は貨幣ではない。
観は数量ではない。

観とは、
**価値の質的重みと流速の積**として機能する
文明的媒介物である。

---

# ETYMOLOGY

```
観（KAN）
　　↑

  ↑
01_ORIGIN / SC-003（OBSERVATION）
```

マノスベ文明OSにおいて、
観測はすべての存在の基盤である（SC-003）。

観（KAN）はその原理を経済単位として具体化したものである。

# PROPERTIES

## 性質1：生成条件（Generation Condition）

観は以下の行為によってのみ生成される：

| 行為 | 生成量の基準 |
|------|------------|
| 身体観測ログの記録（CL-001） | 深度 × 正直さ |
| 場の観測・状態記録 | 精度 × 非評価性 |
| 他者への伝達・共有 | 受容者の状態変化量 |
| ファシリテーション | 参加者の循環促進量 |
| 新たな接続の生成 | 接続の持続性 |

観は**行為の量**ではなく**観測の質**によって決まる。
1000回の表面的記録より、1回の深い観測が多くの観を生む。

---

## 性質2：減衰特性（Decay Property）

観は循環しないと減衰する。

```
観の残存量 = 初期観量 × e^(-λt)
　　　　　　λ = 固定化係数（循環しないほど大きくなる）
　　　　　　t = 最終循環からの経過時間
```

未共有の観測ログ、
配布されない知識、
伝わらないファシリテーション技術は
観を減衰させる。

**観は流れることで保存される。**

---

## 性質3：変換可能性（Convertibility）

観は以下のものと変換できる：

```
観 ←→ ファシリテーション時間
観 ←→ 場へのアクセス権
観 ←→ 知識プログラム
観 ←→ 観測支援（AI・人間）
観 ←→ 共同設計への参加権
```

観は法定通貨（円）と**直接交換されない**。
法定通貨との接続は13_MONETIZATIONが担う。

---

## 性質4：非独占性（Non-Monopoly）

観は独占できない。

ひとつの観測から生成された観は、
共有されることで複数の参加者に分配されるが、
元の生成者の観は減少しない。

```
共有前：生成者が観 × 10 を保有
共有後：生成者が観 × 10 + 受取者が観 × 8（新たに生成）
```

**知識の共有は観を増殖させる。**
これが文明経済の根本的な豊かさの源泉である。

---

## 性質5：観測可能性（Observability）

観は観測可能でなければ存在しない。

主観的な「貢献した気持ち」は観にならない。
他者によって観測・確認された状態変化のみが観を生む。

これは文明OSの観測原理（SC-003）の経済的適用である。

# GENERATION_EXAMPLE

実際の観の生成例：

| 行為 | 生成観量の目安 |
|------|-------------|
| 身体観測ログ1件（深度：浅） | 0.1〜0.3 KAN |
| 身体観測ログ1件（深度：深） | 1〜3 KAN |
| 他者の状態変化を正確に記録 | 2〜5 KAN |
| ワークショップのファシリテーション（10名） | 10〜50 KAN |
| 文明ログの新たな構造発見と共有 | 20〜100 KAN |
| 組織の文明OS実装支援（3ヶ月） | 500〜2,000 KAN |

これらは固定値ではなく、
**受取者側の状態変化**によって変動する。

# MULTI_LAYER_CURRENCY_SYSTEM

観（KAN）は文明経済の**統合層通貨**として機能する。
各層には固有の通貨が存在し、観はそれらを統合する。

```
┌─────────────────────────────────────┐
│  L5: 文明層  ── 観（KAN）            │ ← 統合通貨
├─────────────────────────────────────┤
│  L4: 社会層  ── 文明度（Civ-Point） │
├─────────────────────────────────────┤
│  L3: 場の層  ── 場の質（Ba-Point）  │
├─────────────────────────────────────┤
│  L2: 関係層  ── 共鳴（Res-Point）   │
├─────────────────────────────────────┤
│  L1: 身体層  ── 気（Ki-Point）      │ ← 最小基盤通貨
└─────────────────────────────────────┘
```

### 各層通貨の定義

**気（Ki-Point）**
身体的エネルギーの循環量。
呼吸の深さ・姿勢の安定・睡眠の質によって変動。

**共鳴（Res-Point）**
他者との真正な接続から生まれる価値。
会話の深さ・相互状態変化の質によって決まる。

**場の質（Ba-Point）**
場の設計・維持・修復から生まれる価値。
参加者の観測密度・自発性の増加量によって決まる。

**文明度（Civ-Point）**
社会・組織・文化への循環的貢献量。
波及範囲 × 持続期間によって決まる。

**観（KAN）**
上記4層の価値を統合する最終媒介。
異なる層の価値を比較・変換する際に使用。

---

# EQUATION

観の生成量 = 観測深度 × 受取者状態変化量 × 自発性係数

観の減衰速度 = 固定化係数 × 未循環時間

観の価値 = 現在保有量 × (1 - 固定化率)

# SUMMARY_FOR_AI

観（KAN）は文明経済の基本単位。
観測行為の質と受取者の状態変化から生成される。
循環しないと減衰し、共有で増殖する非独占通貨。
法定通貨との直接交換は禁止（13_MONETIZATION経由）。
気・共鳴・場の質・文明度の4層通貨を統合する。

# SUMMARY_FOR_HUMAN

「観（かん）」とは、
この文明のエネルギーの単位です。

あなたが本当に観察したとき、
あなたが誰かの変化を正直に記録したとき、
あなたが場を整えて誰かが動けるようにしたとき、

観が生まれます。

でも、観は溜め込むと消えていきます。
誰かに渡したとき、
誰かと共有したとき、
観は増えていきます。

これが文明経済の基本です。
