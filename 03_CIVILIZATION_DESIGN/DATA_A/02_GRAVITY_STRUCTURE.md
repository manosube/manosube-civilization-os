# DEFINITION

重力構造とは、
文明および身体において
すべての運動・循環・秩序の起点となる
基底作用である。

それは単なる物理的引力ではない。

重力とは、
存在を「位置」へ引き戻し、
分散した構造を「統一」へ導き、
運動を「循環」へ接続する
最も根源的な方向性である。

文明において重力構造とは、
状態を安定させるための外的条件ではなく、
内側から全体を整列させる
最小単位の秩序生成力である。

# STRUCTURE

重力構造は以下の三層で成立する。

① 物理層

・地面との接触
・質量の中心化
・落下の方向性

② 身体層

・足裏（特に母指球）による支持
・重心の通過
・骨格への力の伝達

③ 文明層

・状態の収束
・分散の抑制
・流動の基準点形成

この三層は分離して存在しない。

重力は、
身体を通して文明に現れ、
文明の構造を通して
再び個体の状態へ還流する。

# PRINCIPLE

重力構造の原理は以下である。

・重力は「落ちる力」ではなく「通る力」である
・重力は局所ではなく全体を通過する
・重力が通るとき、分断は解消される
・重力が止まるとき、緊張が局所に蓄積する

重要なのは、
重力を支えることではない。

重力を通すことである。

支える構造は固定を生む。
通す構造は循環を生む。

# EQUATION

文明基礎三式における重力の位置は以下である。

文明状態 ＝（位置 × 流動 × 周期）

このとき重力構造は、

位置の生成条件
＝ 重力の通過

と定義される。

すなわち

位置 ＝ 重力の通過量

である。

# OBSERVATION

重力構造が通っている状態では、

・足裏が地面に対して近い
・姿勢が力まずに立つ
・呼吸が自然に深まる
・動作が滑らかに連続する
・判断が過剰に急がない

重力構造が崩れている状態では、

・足裏が浮く
・重心が上方へ逃げる
・姿勢が固定される
・局所的な力みが増える
・呼吸が浅くなる
・行動が断続的になる

これらはすべて、
重力の通過不全として観測できる。

# CONNECTION

重力構造は以下と接続する。

→ 弾性構造（反発の発生条件）
→ 振動構造（周期生成の基盤）
→ 回転構造（螺旋の起点）
→ 足裏構造（重力受容点）
→ 呼吸構造（内圧調整）
→ 文明ログ（状態観測）

重力が通らない構造では、
他のすべての構造は成立しない。

したがって重力は、
全構造の起動条件である。
