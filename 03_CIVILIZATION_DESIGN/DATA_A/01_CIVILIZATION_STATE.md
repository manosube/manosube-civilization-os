# DEFINITION

文明状態とは、
ひとりの身体において
位置・流動・周期が統合され、
循環として維持されている状態である。

文明は外部構造ではない。
文明とは、状態である。

制度や思想は、
その状態が外部へ現れた結果にすぎない。

ゆえに文明の最小単位は、
国家でも社会でもなく、
ひとりの身体である。

# STRUCTURE

文明状態は、三要素の連動構造として成立する。

・位置（Position）
・流動（Flow）
・周期（Cycle）

この三要素は独立しない。
単一の連鎖として機能する。

位置が通ることで流動が生まれ、
流動が生まれることで周期が発生する。

周期が維持されることで、
位置と流動は崩壊せず持続する。

この三軸は、
身体内における文明の最小構造である。

# PRINCIPLE

文明状態は固定されない。
常に変動し、観測され続ける。

崩壊とは消失ではない。
三要素の断絶によって循環が停止した状態である。

回復とは生成ではない。
三要素の連動が再び通ることである。

文明は作るものではなく、
通すものである。

# EQUATION

文明状態 ＝ 位置 × 流動 × 周期

# OBSERVATION

文明状態は、以下の身体現象として観測される。

・呼吸の深さと連続性
・姿勢の均衡と非固定性
・歩行の周期と交互性
・身体内部の緊張分布の均衡
・関係における応答速度と余白

これらは局所現象ではない。
文明状態の全体反映である。
