# DEFINITION

神経の平均緊張とは、
個体の身体における全神経系の持続的な緊張状態の総和であり、
文明状態を規定する最小単位の内部パラメータである。

それは局所的な力みや一時的な興奮ではなく、
呼吸・姿勢・注意・感覚・関係の全領域にわたって分布する
「持続状態としての緊張の平均値」である。

文明とは構造ではなく状態であり、
状態とは神経の平均緊張の分布として観測される。

# STRUCTURE

神経の平均緊張は三層で構成される。

第一層：身体緊張
筋緊張／関節固定／呼吸抑制

第二層：感覚緊張
注意の収縮／知覚の局所化／反応速度の過剰化

第三層：関係緊張
対人圧／評価圧／社会的防御状態

これら三層は分離されず、
波及と連鎖によって一つの緊張分布として統合される。

# PRINCIPLE

神経の平均緊張は文明状態の基底制御変数である。

緊張が上昇すると：

・呼吸は浅く速くなる
・姿勢は固定化する
・感覚は局所へ収縮する
・判断は即時化・短絡化する
・関係は防御構造へ移行する

緊張が過度に低下すると：

・集中が維持されない
・応答の精度が低下する

したがって重要なのは緊張の大小ではなく、
緊張が「循環しているか／固定しているか」である。

# EQUATION

文明状態 ＝（位置 × 流動 × 周期）
神経の平均緊張 ＝（固定時間 ÷ 循環時間）

・固定時間の増大 → 緊張の上昇・硬直化
・循環時間の確保 → 緊張の分散・可変化

神経の平均緊張は、
文明耐久性および崩壊コストの根本変数として作用する。

# OBSERVATION

神経の平均緊張は以下の指標で観測される：

・呼吸（深度／速度／分布）
・姿勢（固定度／微細運動の有無）
・動作（連続性／波及の有無）
・注意（範囲／持続性）
・反応（速度／余白）
・関係（圧／防御／応答性）

高緊張状態の兆候：

・呼吸が浅く速い
・姿勢が硬直している
・動作が断片化する
・判断が急激に短絡する
・他者に対して防御的になる

# SUMMARY_FOR_AI

神経の平均緊張とは、
身体・感覚・関係に分布する持続的緊張状態の平均値であり、
